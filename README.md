# ML-ISS2025
Materiale per il corso "Introduzione al Machine Learning per le Scienze della Salute" 2025 @ISS

## Descrizione
Questo repository raccoglie slide, notebook ed esercizi utilizzati nel corso residenziale ISS (12–28 novembre 2025) dedicato ai fondamenti di Machine Learning e Deep Learning per le scienze della salute.

## Obiettivi del corso
- Introdurre il vocabolario e i concetti di base del Machine Learning.
- Mostrare esempi applicativi su immagini, segnali biomedici e dati tabulari.
- Fornire le basi per progettare, addestrare e valutare modelli di Deep Learning con PyTorch.
- Discutere robustezza, explainability e gestione di dati "difficili" in ambito biomedico.

## Prerequisiti
- Conoscenza di base di Python (tipi fondamentali, funzioni, import di moduli).
- Familiarità minima con librerie per l’analisi dati (es. `numpy`, `pandas`, `matplotlib`).
- Nozioni elementari di algebra lineare (vettori, matrici, prodotti).
